# api-gateway

Lightweight API gateway for microservices

## Installation

```bash
npm install
```

## Usage

```javascript
const apiGateway = require('./api-gateway');

// Create an instance
const instance = new ApiGateway();

// Example usage
console.log(instance);
```

See `src/index.js` for full API documentation.
